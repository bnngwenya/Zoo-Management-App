# 🦁 Zoo Management App

A modern, full-featured zoo management application built with React. Schedule visits, explore animal profiles, and manage zoo operations with ease.

![React](https://img.shields.io/badge/React-18.2-blue)
![React Router](https://img.shields.io/badge/React_Router-6.20-red)
![Vite](https://img.shields.io/badge/Vite-5.0-purple)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Features

- 📊 **Interactive Dashboard** - Real-time statistics and activity feed
- 🦁 **Animal Catalog** - 15+ detailed animal profiles with images and fun facts
- 🎫 **Smart Booking System** - Multi-step form with real-time availability
- 🌓 **Dark/Light Mode** - Theme persistence with localStorage
- 🔍 **Search & Filter** - Find animals by name, species, or category
- 📱 **Fully Responsive** - Optimized for all devices

## 🛠️ Tech Stack

- **Frontend Framework:** React 18
- **Routing:** React Router 6
- **Styling:** CSS Modules
- **Build Tool:** Vite
- **State Management:** Context API + Custom Hooks
- **Storage:** LocalStorage
- **Icons:** React Icons

## 🚀 Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/bnngwenya/zoo-management-app.git
   cd zoo-management-app

2. Install dependencies
bash
npm install

3. Start development server
bash
npm run dev

4. Build for production
bash
npm run build

5. Preview production build
bash
npm run preview
